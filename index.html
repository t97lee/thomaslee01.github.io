Projects

Generating Pressure Graphs via Antoine's Equation

A web application that generates graphs of vapour pressure (mmHg) vs. temperature (°C) from Antoine's Equation. The app prompts the user to input a temperature in degrees celsius (°C) and the coefficients, A, B, and C for the equation and outputs an interactive graph generated from Bokeh. Additionally, users may request a table of values to be generated with just a click of a button.

The project is developed using Python for the calculations and data processing and Streamlit was used for the front-end. It is currently a WIP and a demo can be found on the GitHub repo here.

Spotify Song Analyser 🎺

Another Streamlit web application. Input a track title of a song on Spotify and obtain features of the song such as:

  • Danceability
  • Energy
  • Acousticness & more

Additionally, there is a feature in which the app can recommend songs based on the inputted track.

The app was written in Python and uses the Spotipy library. The data is then put into a Pandas DataFrame and graphed using Pandas Chart Visualization. All source code is available on GitHub here.

Arduino RTD Temperature Sensor

This is a project in which an Arduino Uno R3 and an RTD (part no. F2222-1000-A-100) was used to determine the cryoscopic constant of a brine solution through the use of circuitry and thermal analysis. Pictures and code are available on the GitHub repo which can be viewed here. More documentation will be uploaded to the repo in the future.
